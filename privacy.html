ClipBot ("the application") is operated by an individual developer, William Hamilton, as a personal, non-commercial project. This policy explains what data the application accesses and how it is used.

What ClipBot does

ClipBot is a single-user automation tool. It has no public sign-in, no accounts, and no end users other than its operator. It:

  • Reads public clip metadata (title, broadcaster name, view count, video file) from Twitch's public Helix API.
  • Processes that video locally/in a private CI pipeline (cropping, captioning, re-encoding).
  • Uploads the resulting video to the operator's own YouTube channel using the YouTube Data API.

No third party ever submits data to ClipBot, and ClipBot does not serve any public-facing product, app, or website that collects visitor data beyond standard, anonymous GitHub Pages hosting logs for this static site.

Google user data

ClipBot authenticates once, via OAuth 2.0, to the operator's own Google account, requesting only the https://www.googleapis.com/auth/youtube.upload scope. This scope is used exclusively to upload videos to the operator's own YouTube channel. ClipBot does not read, modify, or delete any other data in the operator's Google Account, and does not access the data of any other Google user.

The OAuth access/refresh token is stored only in a local file on the operator's own machine and as an encrypted secret in the operator's private GitHub Actions environment. It is never transmitted to, or accessible by, any third party.

Any use and transfer of information received from Google APIs to any other app will adhere to the Google API Services User Data Policy, including the Limited Use requirements.

Data retention and deletion

The operator can revoke ClipBot's access at any time from Google Account → Security → Third-party access, which immediately invalidates the stored token. To request deletion of any data related to this application, contact the email address below.

Twitch data

ClipBot only reads public clip information exposed by Twitch's Helix API (clip title, broadcaster name, view count, game category, video file). It does not access private Twitch account data, does not require a Twitch login, and does not use Twitch user OAuth scopes.

Streamer takedown requests

Any streamer who would prefer their clips not be featured can email the address below at any time; the request will be honored the same day.

Children's privacy

ClipBot is not directed at children and does not knowingly collect data from anyone. It has no sign-up flow or public user accounts of any kind.
